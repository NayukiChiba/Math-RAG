# dataStat - 数据统计模块

对术语数据进行统计分析与可视化，评估数据质量。

## 模块结构

```
dataStat/
├── chunkStatistics.py   # 统计分析与可视化
└── __init__.py
```

## chunkStatistics.py

对 `data/processed/chunk/` 下的所有术语 JSON 文件进行全面统计。

**统计维度**：

| 维度 | 内容 |
|------|------|
| 术语规模 | 总数、按书籍分布、按学科分布 |
| 字段覆盖率 | 各字段的填充率（definitions、formula、usage 等） |
| 长度分布 | 定义文本长度、公式数量、相关术语数等 |
| 质量指标 | 缺失字段、低质量条目等 |

**生成可视化图表**（6 张）：
1. 术语学科分布（饼图）
2. 定义数量分布（柱状图）
3. 公式数量分布（柱状图）
4. 定义文本长度分布（直方图）
5. 字段覆盖率（横向柱状图）
6. 各书籍术语数量对比（柱状图）

**使用方法**：

```bash
python dataStat/chunkStatistics.py
```

**输出**：

| 文件 | 说明 |
|------|------|
| `data/stats/chunkStatistics.json` | 结构化统计报告 |
| `data/stats/visualizations/*.png` | 可视化图表（6 张，300 DPI） |

**依赖**：`matplotlib`（可选，无则跳过图表生成）
