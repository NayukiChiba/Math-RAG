# runner.py

## 概述
`evaluationData/queryGeneration/runner.py` 作为造测试用 query 的流水线流程大管家或者是顶端状态控制机模块引擎。它串联起了接收来自 `cli.py` 的解析完的环境变量/配置文件参数，进而初始化读取语料源库或是部分文本，设置好了生成器的生成规则或者是重试回退机制以及多并发进程配置，随后控制开始循环或者批量跑，然后再在恰当断点调度 `ioOps` 落库文件并捕捉全部运行中异常、统计生成速率与质量打点日志反馈。

### 核心方法
#### `async def generate_batch(...)` 或 `def run(...)`
异步或同步调度整批文本交送预测，跟踪成功生成比例并维护断点记录逻辑封装引擎处。
