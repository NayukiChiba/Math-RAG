# evaluator.py

## 概述
`modelEvaluation/quickEvalCore/evaluator.py` 它是对整条或者是某个简单的单独检索或者推演端，不用走极度冗长沉重的上万条测试对的庞大且需跑几十个小时的管线，而是只需要抽样一个很小或者具有极其代表价值以及特定问题测试（比如一百条快速看看今天修改的分块算子效果如何的）的轻量化评估判断规则匹配或核心打分的包裹体类引擎操作控制台实现。

## 典型子接口配置点

### `class QuickEvaluator(...)`
利用比如极简或者是本地模型，在不需要发包请求的轻快条件下提供针对测试流向一个微缩版集的测试打分组件包裹管理功能代码块。
