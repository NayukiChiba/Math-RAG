# hybridPlus.py

## 概述
`retrieval/retrieverModules/hybridPlus.py` 作为基础多路召回（混合检索）的扩展版本封装核心部件块逻辑结构执行点。除了包含常规向量与字面的混并（交集或并集融合加权），通常它能再外挂诸如专门针对数学图表/LaTeX 公式/图像区域的特定额外评分管道，或是通过一个独立的 `Reranker` 组件将原本简单混并完的产出列表再推入一个利用如交叉编码器等非常昂贵但精准的模型进一步作精确定位排名的复合式流程的总调动封装机制层控制接口引擎实现包。

## 核心装配

### `class HybridPlusRetriever(HybridRetriever)`
一种基于混合组装管线派生的更为复杂多维度的复合式超级检索门面对象机制；例如允许载入预过滤图谱或者前置关键词剔除等多阶流程以及针对大块文本引入特定增强因子的总包管理引擎结构。
